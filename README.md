# ♻️ Recycling-App  

### Clone & run

```bash
git clone git@github.com:yildiz-fatih/recycling-app.git
cd recycling-app

# install JS dependencies
npm install         

# start local dev server on http://localhost:8081
npm start

# creates public/bundle.js + public/index.html
npm run build

# or: npx cap sync ios / android
npx cap sync

# open the Xcode workspace
npx cap open ios
```
